This project visualizes demographic and occupational data of study participants across Ontario regions.

üìç Geographic Data

The map is based on a custom GeoJSON file created in QGIS to define five Ontario regions: Central (CO), South Central (SC), South West (SW), North West (NW), Eastern Ontario (EO).

The region codes from the GeoJSON are matched with the participant dataset via the Code_Region field.

Note: For this version, only a static snapshot of the dashboard is shown. The full map was implemented in Power BI and QGIS using this GeoJSON structure.
